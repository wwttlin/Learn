## 🚀 快速開始

### 1. 首次部署

```bash
# Ubuntu 伺服器
chmod +x 簡易部署.sh
./簡易部署.sh
```

### 2. 配置 Nginx

```bash
chmod +x setup-nginx.sh
./setup-nginx.sh
```

### 3. 遇到問題

```bash
chmod +x 一鍵修復.sh
./一鍵修復.sh
```

## 💡 為什麼要清理？

1. **減少混淆** - 太多相似的檔案讓人不知道用哪個
2. **提高效率** - 只保留最好用的工具
3. **易於維護** - 更少的檔案更容易管理
4. **清晰結構** - 一目了然的檔案組織
