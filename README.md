# 補習班電腦管理系統

一個完整的補習班管理系統，包含學生管理、課程管理和繳費管理功能。

## 功能特色

### 📊 系統總覽
- 即時統計數據顯示
- 學生數量、課程數量、月收入統計
- 最近繳費記錄和待處理事項

### 👥 學生管理
- 新增、查看學生資料
- 記錄學生和家長聯絡資訊
- 學生狀態管理（在學/停學）

### 📚 課程管理
- 建立和管理課程
- 設定不同繳費方式的價格（月繳、季繳、半年繳）
- 課程描述和詳細資訊

### 💰 繳費管理
- 記錄學生繳費情況
- 支援多種繳費方式（月繳、季繳、半年繳）
- 繳費狀態追蹤（已繳費、待繳費、逾期）

## 技術架構

- **前端**: React + TypeScript + Tailwind CSS
- **後端**: Node.js + Express
- **資料庫**: SQLite
- **開發工具**: 熱重載、並行開發

## 安裝與執行

### 1. 安裝依賴套件

```bash
# 安裝後端依賴
npm install

# 安裝前端依賴
cd client
npm install
cd ..
```

### 2. 啟動開發環境

```bash
# 同時啟動前端和後端
npm run dev
```

或者分別啟動：

```bash
# 啟動後端 API 服務 (Port 5000)
npm run server

# 啟動前端開發服務 (Port 3000)
npm run client
```

### 3. 訪問系統

- 前端界面: http://localhost:3000
- 後端 API: http://localhost:5000

## 資料庫結構

系統會自動建立以下資料表：

- `students` - 學生資料
- `courses` - 課程資料
- `student_courses` - 學生課程關聯
- `payments` - 繳費記錄

## 使用說明

### 初次使用
1. 先在「課程管理」中建立課程
2. 在「學生管理」中新增學生資料
3. 在「繳費管理」中記錄學生繳費

### 日常操作
- 查看系統總覽了解整體狀況
- 定期更新學生繳費記錄
- 管理課程價格和內容

## 開發說明

### 專案結構
```
├── server/           # 後端 API
│   └── index.js     # 主要伺服器檔案
├── client/          # 前端 React 應用
│   └── src/
│       ├── components/  # React 組件
│       └── App.tsx     # 主應用程式
├── package.json     # 後端依賴
└── tutoring.db      # SQLite 資料庫檔案
```

### API 端點
- `GET/POST /api/students` - 學生管理
- `GET/POST /api/courses` - 課程管理  
- `GET/POST /api/payments` - 繳費管理
